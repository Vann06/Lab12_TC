Laboratorio de Teoría de Computación - Lab 12


Contenido
- `problema1/Laboratorio_12_TC.PDF` — ejercicio 1 en papel
- `problema2/p2.js` — Ordena una lista de objetos (por modelo, color y marca).
- `problema3/p3.js` — Trasponer una matriz (filas <-> columnas).
- `problema4/p4.js` — Filtrar elementos de una lista usando `Set` + `filter`.

Cómo ejecutar
Abre una terminal en la raíz del proyecto y ejecuta:

node problema2/p2.js
node problema3/p3.js
node problema4/p4.js
